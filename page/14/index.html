1. Introduction to Nyquist Criterion

==Mapping== \[ F(s)=s^2+2s+1 \\ s=4+j3 \] s-plane to F-plane \[ F(4+j3)=16+j30 \] ==Contour==

Consider a collection of points, called contour A.

Map A into Contour B with function F. \[ F(s)=\frac{(s-z_1)(s-z_2) \dotsm}{(s-p_1)(s-p_2) \dotsm} \] We assuming clockwise direction for mapping.

If contour A enclose one zero, then contour B enclose one origin in a clockwise direction.

If contour A enclose one pole, then contour B enclose one origin in a counterclockwise direction.

If contour A enclose one zero and one pole, then contour B does not enclose origin.

Letting \[ G(s)=\frac{N_G}{D_G} \quad H(s)=\frac{N_H}{D_H} \] for the open loop system, we have \[ G(s)H(s)=\frac{N_G N_H}{D_GD_H} \] for the closed loop system, we have \[ T(s)=\frac{G(s)}{1+G(s)H(s)} \\ =\frac{N_GD_H}{D_GD_H+N_GN_H} \] and we have \[ \tag{1.1} 1+G(s)H(s)=\frac{D_GD_H+N_GN_H}{D_GD_H} \] for (1.1), we find the poles are the poles of closed loop system and the zeros are the poles of the open loop system.

if we have known the number of unstable poles of the open loop system (poles of (1.1) in the right plane) and denote it with P. 3. Application Domain

The first important class consists of "almost" linear system. e.g. the system with a linear plant and a linear controller but the actuator is nonlinear.

The second important class is the genuinely nonlinear system like the example.

4. Basic Assumption

  1. there is only a single nonlinear component
  2. the nonlinear component is time invariant
  3. corresponding to a sinusoidal input, only the fundamental output is considered
  4. the nonlinear is odd

1 implies that if there are two nonlinear parts. they should be lumped together or one can be neglected.

2 implies that we only consider only autonomous nonlinear system

3 implies that the linear part should be a low pass filter

4 means that the nonlinearity relation f(x) between input and output is symmetric about the origin for the simplicity of Fourier expansion of the output.
