3. Convex Functions

3.1 Basic properties and examples

3.1.1 Definition

A function \(f: \mathbb R^n\rightarrow \mathbb R\) is convex if \(\text{dom} f\) is a convex set and if for all \(x,y\in\text{dom} f\), and for \(0\le\theta\le 1\) we have \[ f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y) \] A function is strictly convex if strict inequality holds when ever \(x\neq y\) and \(0<\theta<1\). We say \(f\) is concave if \(-f\) is convex. For an affine function, we always have the equality of the above definition, so all affine functions are both convex and concave. Conversely, any function that is convex and concave is affine.

A function is convex if and only if it is convex when restricted to any line that intersects its domain. In other words \(f\) is convex if and only if for all \(x\in \text{dom} f\) and all \(v\), the function \(g(t)=f(x+tv)\) is convex.

3.1.2 Extended-value extensions

It is often convenient to extend a convex function to all of \(\mathbb R^n\) by defining its value to be infinity outside its domain. \[ \tilde f(x)= \begin{cases} f(x)& x\in \text{dom}\space f\\ \infty& x\notin \text{dom}\space f \end{cases} \] The extension is defined on all \(\mathbb R^n\), the extension can simplify notation since we do not need to explicitly describe the domain.

Example (Indicator function of a convex set)

Let \(C\subseteq \mathbb R^n\) be a convex set, and consider the convex function \(I_C\) with domain \(C\) and \(I_C(x)=0\space \forall x\in C\). In other words, the function is identically zero on the set \(C\). Its extended value extensions is given by \[ \tilde I_C(x)= \begin{cases} 0 & x\in C\\ \infty & x\notin C \end{cases} \] We can play several notation tricks with the indicator function \(I_C\). For example, the problem of minimizing a function \(f\) on the set \(C\) is the same as minimizing the function \(f+\tilde I_C\) over all of \(\mathbb R^n\).

3.1.3 First-order conditions

Suppose \(f\) is differentiable, Then \(f\) is convex if and only if \(\text{dom} f\) is convex and \[ f(y)\ge f(x)+\nabla f(x)^T(y-x) \] holds for all \(x,y \in \text{dom} f\).

Strict convexity can also be characterized by a first-order condition. \[ f(y)> f(x)+\nabla f(x)^T(y-x) \] Proof

First we prove the \(n=1\) case.

We show that a differentiable function is convex if and only if \[ f(y)\ge f(x)+f'(x)(y-x) \] for all \(x,y \in \text{dom} f\).

To prove the necessity, assume \(f\) is convex and \(x,y \in \text{dom} f\). Since \(\text{dom} f\) is convex, we have for all \(0<t\le 1\), \(x+t(y-x) \in dom \space f\), since \(f\) is convex, we have \[ f(x+t(y-x))\le tf(y)+(1-t)f(x) \] Then we have \[ f(y)\ge\frac{f(x+t(y-x))-f(x)}{t}+f(x) \] Taking the limit and we have the theorem.

To show the sufficiency, assume all \(x,y \in \text{dom} f\). Choose any \(x\neq y\) and \(0\le \theta\le 1\) and let \(z=\theta x+(1-\theta)y\) \[ f(x)\ge f(z)+f'(z)(z-x)\quad f(y)\ge f(z)+f'(z)(z-y) \] Multiplying the first inequality with \(\theta\) and the second with \(1-\theta\) and adding them, then we get the proof.

The same for general case.

3.1.4 Second-order conditions

We now assume that \(f\) is twice differentiable, that is, its Hessian or second order derivative \(\nabla^2 f\) exists at each point in \(\text{dom} f\). Then \(f\) is convex if and only if \(\text{dom} f\) is convex and its Hessian is positive semi definite: \[ \nabla^2 f(x)\succeq 0\quad \forall x\in dom \space f \] if \(\nabla^2 f(x)\succ 0\quad \forall x\in dom \space f\), then \(f\) is strictly convex. The strictly convex converse is not true. i.e. \[ f(x)=x^4 \] is strict convex but it has zero second derivative at \(x=0\).

Example (Quadratic functions)

Consider the quadratic function \(f:\mathbb R^n\rightarrow \mathbb R\), with \(\text{dom} f=R^n\), given by \[ f(x)=\frac{1}{2}x^TPx+q^T x+r\\ \nabla^2 f(x)=P\quad \forall x \] That means \(f\) is convex if and only if \(P\succeq0\).

Remark The theorem requirement that \(\text{dom} f\) be convex cannot be dropped. For example, \[ f(x)=\frac{1}{x^2} \] is not convex since the \(\text{dom} f\) is not convex, but \(f''(x)>0\)

3.1.5 Examples

  1. Exponential. \(e^{ax}\) is convex on \(\mathbb R\) for any \(a\in \mathbb R\).

  2. Powers. \(x^a\) is convex on \(\mathbb R_{++}\) when \(a\ge 1\) or \(a\le 0.\)

  3. Powers of absolute value. \(|x|^p\) is convex on R for \(p\ge1\).

  4. Logarithm. log x is concave on \(\mathbb R_{++}\).

  5. Negative entropy. $x x $ is convex on \(\mathbb R_{++}.\)

\[ f''(x)=\frac{1}{x}>0 \]

  1. Norm. Every norm on \(\mathbb R^n\) is convex. \(f:\mathbb R^n\rightarrow \mathbb R\) is a norm and \(0\le\theta\le1\), then

\[ f(\theta x+(1-\theta)y)\le f(\theta x)+f((1-\theta)y)=\theta f(x)+(1-\theta)f(y). \]

  1. Max function is convex on \(\mathbb R^n\) . \(f(x)=\max_i x_i\) and \(0\le \theta\le 1\)

\[ f(\theta x+(1-\theta)y)=\max_i(\theta x_i+(1-\theta)y_i)\le\max_i(\theta x_i)+\max((1-\theta)y_i)=\theta f(x)+(1-\theta)f(y). \]

  1. Quadratic-over-linear function. The function \(f(x,y)=\frac{x^2}{y}\) is convex on \(\mathbb R\times \mathbb R_{++}\).

    To show the function is convex, we have

\[ \nabla^2f(x,y)= \begin{bmatrix} \frac{2}{y} & -\frac{2x}{y^2} \\ -\frac{2x}{y^2} & \frac{2x^2}{y^3} \end{bmatrix}=\frac{2}{y^3} \begin{bmatrix} y^2 & -xy\\ -xy & x^2 \end{bmatrix}\succeq0. \]

  1. Log-sum-exp. The function \(f(x)=\log(e^{x_1}+\dotsm+e^{x_n})\) is convex on \(\mathbb R^n\).

    Define \(z=(e^{x_1},e^{x_2},\dotsm,e^{x_n})\), then \(f(x)=\log (1^T z)\). \[ \frac{\partial f(x)}{\partial x_i}=\frac{1}{1^Tz}z_i\\ \frac{\partial^2 f}{\partial x_i\partial x_j}= \begin{cases} -(1^Tz)^{-2}z_jz_i+(1^Tz)^{-1}z_{i=j} & i=j\\ -(1^Tz)^{-2}z_jz_i & i\neq j \end{cases}. \] That means \[ \nabla^2 f(z)=-(1^Tz)^{-2}zz^T+(1^Tz)^{-1}\text{diag}(z). \] To check if it is positive definite \[ v^T\nabla^2 f(z) v=-(1^Tz)^{-2}v^Tzz^Tv+(1^Tz)^{-1}v^T\text{diag}(z)v\\ =(1^Tz)^{-2}[(1^Tz)v^T\text{diag}(z)v-v^Tzz^Tv]\\ =(1^Tz)^{-2}\left[\left(\sum_{i=1}^n z_i \right) \left(\sum_{i=1}^n v_i^2z_i\right)-\left(\sum_{i=1}^n v_iz_i\right)^2\right]. \] By Cauchy-Schwarz inequality \((a^Ta)(b^Tb)\ge(a^Tb)^2\), where \(a=\sqrt z_i\) and \(b=v_i\sqrt z_i\), we can know the above matrix is positive definite.

  2. Geometric mean. \(f(x)=(\prod_{i=1}^n x_i)^{1/n}\) is concave on \(\text{dom} f=\mathbb R^n_{++}\). \[ \frac{\partial f(x)}{\partial x_i}=\frac{1}{n}(\prod_{k=1}^n x_k)^{1/n-1} \frac{\prod_{k=1}^{n} x_k}{x_i}= \frac{1}{n x_i}(\prod_{k=1}^n x_k)^{1/n}\\ \frac{\partial f^2(x)}{\partial x_i\partial x_i}=(1-n)\frac{1}{(nx_i)^2} (\prod_{k=1}^n x_k)^{1/n}\\ \frac{\partial f^2(x)}{\partial x_i\partial x_j}=\frac{1}{n^2x_ix_j}(\prod_{k=1}^n x_k)^{1/n}. \] Then we have \[ \nabla^2f(x)=\frac{1}{n^2}(\prod_{k=1}^n x_k)^{1/n}\left(qq^T-n\space diag(1/x_1^2,\dotsm,1/x_n^2)\right) \] where \(q_i=\frac{1}{x_i}\).

Then for any \(v\), we have \[ v^T \nabla^2 f(x)v =\frac{1}{n^2}(\prod_{k=1}^n x_k)^{1/n}\left(\left(\sum v_iq_i\right)^2-n\sum {v_i^2q_i^2} \right)\le 0. \]

  1. Log-determinant. For the function \(f(X)=\log \det X\) is concave.

    We can verify concavity by considering an arbitrary line, given by \(X=Z+tV\) where \(Z,V\in \mathbb S^n\). \[ g(t)=f(Z+tV)=\log \det (Z+tV)\\ =\log\det(Z^{1/2}(I+tZ^{-1/2}VZ^{-1/2})Z^{1/2})\\ =\log\det(I+tZ^{-1/2}VZ^{-1/2})+\log \det Z\\ =\sum_i\log(1+t\lambda_i)+\log \det Z\\ g'(t)=\sum_i\frac{\lambda_i}{1+t\lambda_i}\\ g''(t)=-\sum_i\frac{\lambda_i^2}{(1+t\lambda_i)^2}<0, \] where \(\lambda_i\) is the eigenvalues of the matrix \(Z^{-1/2}VZ^{-1/2}\). Therefore, concave.

3.1.6 Sublevel sets

The \(\alpha\)-sublevel set of a function \(f:\mathbb R^n\rightarrow \mathbb R\) is defined as \[ C_\alpha=\{x\in \text{dom}\space f\;|\;f(x)\le\alpha\} \] The sublevel set of a convex function is convex.

3.1.7 Epigraph

The graph of a function \(f:\mathbb R^n\rightarrow \mathbb R\) is defined as \(\{(x,f(x))\;|\;x\in \text{dom} f\}\).

The epigraph (epi means above) of a function \(f:\mathbb R^n\rightarrow \mathbb R\) is defined as \[ \text{epi} f=\{(x,t)\;|\;x\in \text{dom} f,f(x)\le t\}. \] The link between convex sets and convex functions is via the epigraph: A function is convex if and only if its epigraph is a convex set.

A function of concave if and only if its hypograph defined as \[ \text{hypo} f=\{(x,t)\;|\;t\le f(x)\} \] is a convex set.

Example (Matrix fractional function)

The function \(f:\mathbb R^n\times \mathbb S^n\rightarrow \mathbb R\) ,defined as \[ f(x,Y)=x^TY^{-1}x \] is convex on \(\text{dom} f =\mathbb R^n\times \mathbb S^n_{++}\) . This function is the generalized quadratic over linear function, \[ \text{epi} f=\{(x,Y,t)\;|\;\space Y\succ 0, x^TY^{-1}x\le t\}\\ =\{(x,Y,t)\;|\;\space Y\succ 0, t-x^TY^{-1}x\ge 0\}. \] By Schur complement \[ S=t-x^TY^{-1}x\ge0\iff X= \begin{bmatrix} Y & x\\ x^T & t \end{bmatrix}\succeq 0. \] We have \[ \text{epi} f=\{(x,Y,t)|\space Y\succ 0, \begin{bmatrix} Y & x\\x^T & t \end{bmatrix}\succeq 0\}. \] The conditions are linear matrix inequality in \((x,Y,t)\) and therefore, $ f $ is convex.

3.1.8 Jensen's inequality and extensions

The basic inequality \[ f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y) \] is sometimes called Jensen's inequality . It is easily extended to convex combinations of more than two points. \[ f(\theta_1x_1+\dotsm+\theta_nx_n)\le\theta_1f(x_1)+\dotsm+\theta_nf(x_n). \] if \(f\) is convex and \(\theta_1+\dotsm+\theta_n =1\) and \(\theta_1,\theta_2,\dotsm,\theta_n\ge0\).

This is also correct in continuous cases \[ f\left(\int_sp(x)xdx\right)\le\int_sf(x)p(x)dx \] if \(f\) is convex, \(p(x)\ge0\),\(\int_sp(x)dx=1\).

In the most general case, we can take any probability measure with support in \(\text{dom} f\) and if \(f\) is convex, we have \[ f(Ex)\le Ef(x). \]

3.1.9 Inequalities

Many famous inequalities can be derived by applying Jensen's inequality to some appropriate convex function.

2. Convex Sets

2.1 Affine and Convex Sets

2.1.1 Lines and line segments

Suppose \(x_1 \neq x_2\) are two points in \(\mathbb R^n\). Points of the form \[ y=\theta x_1+(1-\theta)x_2, \] where \(\theta\in \mathbb R\), form a line passing \(x_1\) and \(x_2\).

If \(0\le\theta\le 1\), the line is restricted between the two points, which is therefore a line segment.

2.1.2 Affine sets

A set \(C \subseteq \mathbb R^n\) is affine if the line through any two distinct points in C lies in C, i.e. for any \(x_1, x_2\in C\) and \(\theta\in R\), we have \(\theta x_1+(1-\theta)x_2 \in C\). We refer to a point of the form \(y=\theta_1 x_1 +\theta_2 x_2 +\dotsm+\theta_k x_k\), where \(\theta_1 + \theta_2+\dotsm+\theta_k=1\) as an affine combination of the points \(x_1,x_2,\dotsm,x_k\). An affine set contains every affine combination of its points.

If \(C\) is an affine set and we have \(x_0 \in C\), then the set \[ V=C-x_0=\{x-x_0|x\in C\}, \] is a subspace (a set that is closed under sums and multiplication).

proof

If \(C\) is an affine set and define \(V=C-x_0\), assume \(v_1, v_2 \in V\), \(\alpha, \beta \in \mathbb R\), and \(\alpha+\beta = 1\), \(x_0\in C\).

Then we have \(v_1 + x_0 \in C\) and \(v_2 + x_0 \in C\).

To complete the proof, we need to prove $v_1+v_2 V $, that means we need to prove \(\alpha v_1+\beta v_2 + x_0 \in C\). \[ \alpha (v_1+x_0)+\beta(v_2+x_0)=\alpha v_1+\beta v_2+(\alpha+\beta)x_0\in C. \] That means \(V\) is a subset. Q.E.D.

Thus an affine set can be expressed as (a subspace plus an offset) \[ C=V+x_0=\{v+x_0|v\in V\}. \] The set of all affine combinations of points in some set \(C\subseteq \mathbb R^n\) is called the affine hull of C, and denoted as \(\text{aff } C\). \[ \text{aff }C=\{\theta_1x_1+\dotsm+\theta_kx_k\;|\; x_1,\dotsm,x_k\in \mathbb R^n,\theta_1+\dotsm+\theta_k=1\} \]

2.1.3 Affine dimension and relative interior

We define the affine dimension of a set \(C\) as the dimension of its affine hull.

We define the relative interior of the set \(C\), denoted as \(\text{relint }C\), as its interior relative to \(\text{aff }C\).

Difference between the interior and relative interior. Consider \(\mathbb R^3\) with the standard Euclidean metric, and look at the set of points \((x,y,0)\) such that \(x^2+y^2\le1\) . A point is in the interior of a set if you can draw a small open ball around it which is itself contained in the set. But for any point in our set, any open ball around any point in it will contain points outside the \(xâˆ’y\) plane, and so it has empty interior. That's technically correct, but it's undesirable behavior. The relative interior of a set is obtained by looking at the affine hull of the set, which in this case is \(\mathbb R^2\), and using the appropriate metric there. So while our set has empty interior, its relative interior is the interior of the unit disc, which is what it "really should be". (From Quora)

We can then define the relative boundary of a set \(C\) as \(\text{cl }C\backslash \text{relint }C\), where \(\text{cl } C\) is the closure of \(C\).

2.1.4 Convex sets

A set \(C\) is convex if the line segment between any two points in \(C\) lies in \(C\).

If for any \(x_1,x_2 \in C\) and any \(\theta\) with \(0\le \theta\le 1\), we have \[ \theta x_1+(1-\theta)x_2\in C. \] We call a point of the form \(\theta_1 x_1+\dotsm+\theta_kx_k\), where \(\theta_1+\dotsm+\theta_k=1\) and \(\theta_i\ge 0\;\forall i\), a convex combination of the points \(x_1,\dotsm,x_k\). As with affine sets, it can be shown that a set is convex if and only if it contains every convex combination of its points.

The convex hull of a set \(C\), denoted as \(\text{conv }C\) or \(\text{co }C\), is the set of all convex combinations of points in \(C\). \[ \text{conv }C=\{\theta_1x_1+\dotsm+\theta_kx_k\;|\; x_1,\dotsm,x_k\in \mathbb R^n,\theta_1+\dotsm+\theta_k=1,\theta_i\ge 0\;\forall i\} \] Attention: every affine set is convex.

The idea of convex combination can be generalized to include infinite sums, integrals and, in the most general form, probability distributions.

Suppose \(p:\mathbb R^n \rightarrow \mathbb R\) satisfies \(p(x)>0\) for all \(x\in C\) and \(\int_Cp(x)dx =1\), where \(C\subseteq \mathbb R^n\) is convex, then \[ \int_Cp(x)x dx\in C. \]

2.1.5 Cones

A set \(C\) is called cone, if for every \(x\in C\) and \(\theta\ge0\) ,we have \(\theta x\in C\).

A set \(C\) is called convex cone if it is convex and a cone, which means that for any \(x_1,x_2 \in C\) and \(\theta_1,\theta_2\ge 0\) (We do not need \(\theta_1+\theta_2=1\) here), we have \[ \theta_1x_1+\theta_2 x_2 \in C. \] We call a point of the form \(\theta_1 x_1+\dotsm+\theta_kx_k\), where \(\theta_i\ge 0\;\forall i\), a conic combination of the points \(x_1,\dotsm,x_k\).

The conic hull of a set \(C\) is the set of all conic combinations of points in C, i.e. \[ \{\theta_1x_1+\dotsm+\theta_kx_k|x_i\in C,\theta_i\ge0,i=1,\dotsm,k\}. \]

We can do a summary here.

We define three kinds of sets and combinations in this section: affine set (combination), convex set (combination), and cone (conic combination).

  1. Affine combination: we need \(\theta_1+\dotsm+\theta_k=1\), but we do not care about \(\theta_i\ge 0\).
  2. Conic combination: we need \(\theta_i\ge 0\), but we do not care about \(\theta_1+\dotsm+\theta_k=1\).
  3. Convex combination: we need \(\theta_i\ge 0\) and \(\theta_1+\dotsm+\theta_k=1\).

2.2 Some Important Examples

  1. The single point set, empty set, and the whole space set are both convex and affine.
  2. Any line is affine, and also hence convex.
  3. A line segment is convex but not affine.
  4. A ray is convex but not affine. It is a cone.
  5. Any subspace is affine, and therefore convex.

2.2.1 Hyperplanes and halfspaces

A hyperplane is the set of the form \[ \{x|a^Tx=b\}, \] where \(a\in \mathbb R^n,a\neq0,b\in \mathbb R\).

Geometrically, \(a\) is the normal vector, \(b\) determines the offset of the hyperplane from the origin. That can be understood by showing the form \[ \{x|a^T(x-x_0)=0\}. \] A hyperplane divides \(\mathbb R^n\) into two halfspaces. A (closed) halfspace is a set of the form, \[ \{x|a^Tx\le b\}. \] Halfspaces are convex but not affine.

The set \(\{x|a^Tx< b\}\) which is the interior of the (closed) halfspace is called an open halfspace.

2.2.2 Euclidean balls and ellipsoids

A Euclidean ball in \(\mathbb R^n\) has the form \[ B(x_c,r)=\{x\;|\;||x-x_c||_2\le r\}=\{x\;|\;\space(x-x_c)^T(x-x_c)\le r^2\} \] A Euclidean ball is obviously a convex set, and it can be proved by, if \(||x_1-x_c||_2\le r\) and \(||x_2-x_c||_2\le r\) and \(0\le\theta\le 1\), then we check the point \(\theta x_1+(1-\theta)x_2\)

\[ ||\theta x_1+(1-\theta)x_2-x_c||_2=||\theta(x_1-x_c)+(1-\theta)(x_2-x_c)||_2\\ \le\theta||x_1-x_c||_2+(1-\theta)||x_2-x_c||_2\\ \le r \] That means the point is also in the set.

A another common representation for the Euclidean ball is \[ B(x_c,r)=\{x_c+ru|\space||u||_2\le1\}. \] A related family of convex sets is the ellipsoids, which have the form \[ \epsilon=\{x|\space(x-x_c)^TP^{-1}(x-x_c)\le r^2\}, \] where \(P\) is symmetric and positive definite, i.e. \(P=P^T\succ 0\).

Another common representation of an ellipsoid is \[ \varepsilon=\{x_c+Au\;|\;||u||_2\le1\}, \] where \(A\) is square and nonsingular. It can be seen that \(A=P^{\frac{1}{2}}\). If \(A\) is singular, then it is a degenerate ellipsoid which is also convex.

2.2.3 Norm balls and norm cones

Suppose \(||\cdot||\) is any norm on \(\mathbb R^n\). From the general properties it can be seen the norm ball given by \[ \{x\;|\;||x-x_c||\le r\}, \] is convex.

The norm cone is given by \[ \{(x,t)\;|\;||x||\le t\} \subseteq \mathbb R^{n+1}, \] is a convex cone.

2.2.4 Polyhedra

A polyhedron is defined as the solution set of a finite number of linear equalities and inequalities, i.e. \[ \mathcal P=\{x\;|\;a_j^Tx\le b_j,j=1,\dotsm,m,c_j^Tx=d_j,j=1,\dotsm,p\}. \] Therefore, a polyhedron is thus the intersection of a finite number of halfspaces and hyperplanes. Affine sets, rays, line segments, and halfspaces are all polyhedra. It is easily shown that polyhedra are convex sets (just use the definition). A bounded polyhedron is called polytope. It is convenient to use the compact notation \[ \mathcal P=\{x|Ax\preceq b,Cx=d\}, \] where \[ A= \begin{bmatrix} a_1^T\\a_2^T\\\vdots\\a_m^T \end{bmatrix} \quad C= \begin{bmatrix} c_1^T\\c_2^T\\\vdots\\c_p^T \end{bmatrix}.\\ \] Simplexes are important family of polyhedra. Suppose \(k+1\) points \(v_0,v_1,\dotsm,v_k\in \mathbb R^n\) are affinely independent that means \(v_1-v_0,v_2-v_0,\dotsm,v_k-v_0\) are linearly independent. Then the simplex is determined by \[ C=\text{conv }\{v_0,\dotsm,v_k\}=\{\theta_0v_0+\dotsm+\theta_k v_k\;|\;\theta\succeq0,1^T\theta=1\}. \] Notation

Roughly speaking, affine independence is like linear independence but without the restriction that the subset of lower dimension the points lie in contains the origin. So three points in space are affinely independent if the smallest flat thing containing them is a plane. They're affinely dependent if they lie on a line (or are the same point).

To describe the simplex as polyhedron, we define \[ y=(\theta_1,\dotsm,\theta_k)\\ B=[v_1-v_0,\dotsm,v_k-v_0]. \] Then \(x\in C\) if and only if \(x=v_0+By\), \(y\succeq0\), and \(1^T y\le 1\).

Then we define \[ A_1B=I\quad A_2B=0. \] Because B is linear independent, we have a nonsingular matrix, \[ A=\begin{bmatrix} A_1\\A_2 \end{bmatrix}\in \mathbb R^{n\times n}. \] Then we have \[ x=v_0+By\\ \iff\\ Ax=Av_0+ \begin{bmatrix} I\\0 \end{bmatrix} y \\\iff \\ \begin{cases} A_1 x=A_1 v_0+y\\ A_2 x=A_2 v_0 \end{cases} \] Then, \(x\in C\) if and only if \(A_2 x=A_2 v_0\), and the vector \(y=A_1x-A_1v_0\) satisfies \(y\succeq 0\) and \(1^Ty\le 1\) , that is same to \[ A_2x=A_2v_0\quad A_1x\succeq A_1v_0\quad 1^TA_1x\le1+1^TA_1v_0 \]

2.2.5 Positive semi-definite cone

The set \(\mathbb S_+^n\) is a convex cone, if \(\theta_1, \theta_2\ge0\) and \(A,B\in \mathbb S_+^n\). Then \(\theta_1 A+\theta_2B \in \mathbb S_+^n\). That can be easily checked by for any \(x\in \mathbb R^n\) , if \(\theta_1,\theta_2\ge0\) and \(A,B\in \mathbb S_+^n\), \[ x^T(\theta_1A+\theta_2B)x=\theta_1x^TAx+\theta_2x^TBx\ge0. \] That means \((\theta_1A+\theta_2B) \in \mathbb S_+^n\).

2.3 Operations That Preserve Convexity

In this section, we describe some operations that preserve convexity of sets, or allow us to construct convex set from others.

2.3.1 Intersection

Convexity is preserved under intersection. If \(S_1\) and \(S_2\) are convex, then \(S_1 \cap S_2\) is convex. Every closed convex set is an intersection of halfspaces.

2.3.2 Affine Function

Recall that a function \(f: \mathbb R^n\rightarrow \mathbb R^m\) is affine if it is a sum of linear function and a constant. That means it has the form \[ f(x)=Ax+b. \] where \(A\in \mathbb R^{m\times n}\) and \(b\in \mathbb R^m\).

Suppose \(S\subseteq \mathbb R^n\) is convex and \(f\) is an affine function. Then the image of \(S\) under \(f\) is convex. Similarly, the inverse image of \(S\) under \(f\) is also convex. This affine function can be considered as a translation of a convex set.

The projection of a convex set onto some of its coordinates is convex. if \(S\subseteq \mathbb R^m\times \mathbb R^n\) is convex, that means the following is convex. \[ T=\{x_1\in \mathbb R^m\;|\;(x_1,x_2)\in S\text{ for some }x_2\in \mathbb R^n\} \] The sum of two convex sets is convex. That means if \(S_1\) and \(S_2\) are convex, the following set is also convex, \[ S_1+S_2=\{x+y\;|\; x\in S_1,y\in S_2\}. \] The partial sum of convex sets is convex. That means if \(S_1\) and \(S_2\) are convex set, the following set is also convex, \[ S=\{(x,y_1+y_2)|\space (x,y_1)\in S_1,(x,y_2)\in S_2\}. \] The Cartesian product of convex sets is convex. That means if \(S_1\) and \(S_2\) are convex set, the following set is also convex, \[ S_1\times S_2=\{(x_1,x_2)\;|\;x_1\in S_1,x_2\in S_2\}. \] Example Polyhedron

The polyhedron can be expressed as the inverse image of the Cartesian product of the nonnegative orthant and the origin under the affine function \(f(x)=(b-Ax,d-Cx)\): \[ \{x\;|\;Ax\preceq b,Cx=d\}=\{x\;|\;f(x)\in \mathbb R^m_+\times\{0\}\}. \] The inverse image of S under the function \(f\) means \[ f^{-1}(S)=\{x\;|\;f(x)\in S\}. \] Example Solution set of linear matrix inequality (LMI), The condition \[ A(x)=x_1A_1+\dotsm+x_nA_n\preceq B \] where \(B,A_i \in \mathbb S^m\), is called the linear matrix inequality (LMI) in \(x\).

The solution set of a LMI \(\{x\;|\;A(x)\preceq B\}\) is convex. Since it is the inverse image of the positive definite cone (2.2.5) under the affine function \(f:\mathbb R^n\rightarrow \mathbb S^m\) given by \[ f(x)=B-A(x). \] That means the solution set is \[ \{x\;|\;f(x)=B-A(x)\in S_+^m\}. \] Example Hyperbolic cone

The set \[ \mathcal N=\{x\;|\;x^T Px\le(c^Tx)^2,c^Tx\ge0\} \] where \(P\in \mathbb S_+^n\) and \(c\in \mathbb R^n\) is convex, since it is the inverse image of the second-order cone \[ \mathcal M=\{(z,t)\;|\;z^Tz\le t^2,t\ge 0 \}, \] under the affine function \(f(x)=(P^{1/2} x,c^Tx)\).

That means the set \(\mathcal N\) should be \[ \{x\;|\;f(x)\in\mathcal M\}=\{x\;|\;(P^{1/2} x,c^Tx)\in\mathcal M\}=\{x\;|\;x^TP^{1/2}P^{1/2}x\le(c^Tx)^2,c^Tx\ge 0\}=\mathcal N. \] Example Ellipsoid

The ellipsoid \[ \varepsilon = \{x|(x-x_c)^TP^{-1}(x-x_c)\le 1\}, \] where \(P\in \mathbb S_{++}^n\), is the image of the unit Euclidean ball \(\{u|\space ||u||_2\le 1\}\) under the affine mapping \[ f(u)=P^{1/2}u+x_c. \] We can check \[ f(x)\in\{P^{1/2}u+x_c|\space||u||_2\le 1\}=\{x|\space||P^{-1/2}(x-x_c)||\le1\}\\=\{x|(x-x_c)^TP^{-1}(x-x_c)\le 1\}=\varepsilon. \]

2.3.3 Linear-fractional and perspective functions

In this section, we explores a class of functions, called linear-fractional, that is more general than affine function but still preserve convexity.

Perspective Function

We define the perspective function \(P:\mathbb R^{n+1}\rightarrow \mathbb R^{n}\), with domain \(\text{dom }P=\mathbb R^n\times \mathbb R_{++}\) as \(P(z,t)=z/t\).

If \(C\subseteq \text{dom } P\) is convex, then its image \[ P(C)=\{P(x)\;|\;x\in C\}, \] is convex.

We consider \((\tilde x,x_{n+1}), (\tilde y,y_{n+1})\in \mathbb R^{n+1}\) with \(x_{n+1}>0\) and \(y_{n+1} > 0\), then for $01 $, we have \[ P(\theta x+(1-\theta)y)=\frac{\theta \tilde x+(1-\theta)\tilde y}{\theta x_{n+1}+(1-\theta) y_{n+1}}=\mu P(x)+(1-\mu)P(y) \] where \(0\le\mu=\frac{\theta x_{n+1}}{\theta x_{n+1}+(1-\theta)y_{n+1}}\le1, P(x)=\frac{\tilde x}{x_{n+1}},P(y)=\frac{\tilde y}{y_{n+1}}\). This also shows that \(P([x,y])=[P(x),P(y)]\).

The inverse image of a convex set under the perspective function is also convex.

Proof is skipped.

Linear-fractional Function

A linear fractional function is formed by composing the perspective function with an affine function.

Suppose \(g:\mathbb R^n\rightarrow \mathbb R^{m+1}\) is affine, where \[ g(x)= \begin{bmatrix} A\\c^T \end{bmatrix}x+ \begin{bmatrix} b\\d \end{bmatrix}. \] The function \(f:\mathbb R^n\rightarrow \mathbb R^m\) given by \(f(x)=p\circ g\), \[ f(x)=\frac{Ax+b}{c^Tx+d}, \] where \(c^Tx+d>0\), is called a linear fractional function, which preserve convexity. If \(C\) is convex and lies in the \(\text{dom }f\), then its image \(f(C)\) is also convex.

Example Conditional probabilities

Suppose \(u\) and \(v\) are random variables that take on values in \((1,\dotsm,m)\) and \((1,\dotsm,n)\) respectively. Let \(p_{ij}\) denote probability of \(u=i, v=j\). Then the conditional probability is \[ \text{prob}(u=i|v=j)=\frac{p_{ij}}{\sum_{k=1}^np_{kj}} \] Denote \(x=[p_{1j},p_{2j},\dotsm,p_{mj}]^T,A=I_{m\times m},b=0,c=1^T_m,d=0\). Thus \(f\) is obtained by a linear fractional mapping from \(p\). Therefore, if the joint probabilities for \(u\) and \(v\) is convex, the conditional probability is also convex.

2.4 Generalized Inequalities

2.4.1 Proper cones and generalized inequalities

A cone \(K\subseteq \mathbb R^n\) is called a proper cone if it satisfies the following

  1. K is convex
  2. K is closed
  3. K is solid, which means it has nonempty interior
  4. K is pointed. It means there are no 2 points inside it which creates a line (not line segment) and the whole line is contained by the cone. (\(x\in K,-x\in K\implies x=0\)) or (\(Q \cap -Q = \{0\}\)).

A proper cone can be used to define a generalized inequality, which is a partial ordering on \(\mathbb R^n\) that has many of the properties of standard ordering on \(\mathbb R\). \[ x\preceq_K y \iff y-x\in K\\ x\prec_K y \iff y-x\in \text{int } K \]

The similar thing for \(\succ_K,\succeq_K\).

Example Nonnegative orthant and componentwise inequality

When \(K=\mathbb R^m_+\). The associated generalized inequality \(x\preceq y\) means that \(x_i\le y_i\) for all elements.

Example Positive semidefinite cone and matrix inequality

When \(K=\mathbb S^n_+\). The associated generalized inequality \(X\preceq Y\) means \(Y-X\) is positive semidefinite.

Properties of generalized inequalities

  1. if \(x\preceq_K y\) and \(u\preceq_K v\), then \(x+u\preceq_K y+v\)
  2. if \(x\preceq_K y\) and \(y\preceq_K z\), then $x_K $ z
  3. if \(\alpha\ge 0\) and \(x\preceq_K y\), then \(\alpha x\preceq_K \alpha y\)
  4. \(x\preceq_K x\)
  5. if \(x\preceq_K y\) and \(y\preceq_K x\), then \(x=y\)
  6. if \(x_i\preceq_K y_i\), \(\lim_{i\rightarrow \infty} x_i =x\) and \(\lim_{i\rightarrow \infty} y_i =y\) , then \(x\preceq_K y\)
  7. if \(x\prec_K y\) then \(x\preceq_K y\)

2.4.2 Minimum and minimal elements

We say that \(x\in S\) is the minimum element of \(S\) (with respect to the general inequality) if for every \(y\in S\) we have \(x\preceq_K y\). A point \(x\in S\) is the minimum element if and only if \[ S\subseteq x+K. \] Something is minimum if it is the unique minimal element. (From Quora)

We say that \(x\in S\) is the minimal element of \(S\) if \(y \in S, y\preceq_K x\) only if \(y=x\). A point \(x\in S\) is the minimal element if and only if \[ (x-K)\cap S=\{x\}. \] Something is minimal if nothing is smaller. (From Quora)

2.5 Separating and Supporting hyperplanes

2.5.1 Separating hyperplane theorem

In this section, we describe an idea that will be important later. The use of hyperplanes or affine functions to separate convex set that do not intersect. The basic result is the separating hyperplane theorem: suppose \(C\) and \(D\) are two convex sets that do not intersect, then there exist $a0 $ and \(b\) such that \(a^Tx\le b\) for all \(x\in C\) and \(a^Tx\ge b\) for all \(x\in D\).

Proof is skipped.

Strict separation is the stronger condition on the separation, which means \(a^Tx<b\;\forall x\in C\) and \(a^Tx>b\;\forall x\in D\).

Converse separating hyperplane theorems

Existence of a separating hyperplane implies that C and D do not intersect is NOT true. Consider the set \(C=D=\{0\}\)

But we have the following theorem. Any two convex sets \(C\) and \(D\), at least one of which is open, are disjoint if and only if there exists a separating hyperplane.

2.5.2 Supporting hyperplanes

Suppose \(C\subseteq \mathbb R^n\), \(x_0\) is a point in this boundary \(\text{bd } C\), i.e. \(x_0\in\text{bd }C=\text{cl }C\backslash\text{int }C\). If \(a\neq0\) satisfies \(a^Tx\le a^T x_0\) for all \(x\in C\) , then the hyperplane \(\{x\;|\;a^Tx=a^T x_0\}\) is called a supporting hyperplane to \(C\) at the point \(x_0\).

Supporting hyperplanes theorem

For any nonempty convex set \(C\), and any \(x_0\in \text{bd } C\), there exists a supporting hyperplane to \(C\) at \(x_0\).

There is also a partial converse of the supporting hyperplane theorem: If a set is closed, has nonempty interior, and has a supporting hyperplane at every point in its boundary, then it is convex.

2.6 Dual Cones and Generalized Inequalities

2.6.1 Dual cones

Let \(K\) be a cone. The set \[ K^*=\{y\;|\;x^Ty\ge0\text{ for all }x\in K\} \] is called a dual cone of \(K\). As the name suggests, \(K^*\) is a cone, and is always convex even when the original cone \(K\) is not. Geometrically, \(y\in K^*\) if and only if \(-y\) is the normal of a hyperplane that supports \(K\) at the origin.

Example Nonnegative orthant. The cone \(\mathbb R_+^n\) is its own dual (self-dual) \[ y^Tx\ge 0\space\forall x\succeq 0 \iff y\succeq 0. \] Example Positive semidefinite cone. We use the standard inner product \[ \langle X,Y\rangle=\text{tr}(X^TY)=\text{tr}(XY)=\sum_{i,j=1}^n X_{ij}Y_{ij}. \] The positive semidefinite cone is self-dual that means for \(X,Y \in \mathbb S^n\) \[ \text{tr}(XY)\ge0 \space\forall X\succeq 0 \iff Y\succeq 0 \] To prove sufficiency

We need to prove if \(Y\notin \mathbb S_+^n\), $ X0(XY)0$.

Suppose \(Y\notin \mathbb S_+^n\). Then there exists \(q\in \mathbb R^n\) with \[ q^TYq=\text{tr}(qq^TY)< 0 \quad\text{trace property: }\text{tr}(AB)=\text{tr}(BA). \] Then let \(X=qq^T\) \[ q^TXYq=tr(q^Tqq^TYq)=q^Tqtr(q^TYq)<0. \] Sufficiency proved.

To prove necessity, we need to prove, if \(Y\succeq 0\), \(\forall X\succeq 0 \;\text{tr}(XY)\ge0\).

Suppose \(X, Y\in \mathbb S_+^n\). Then we can express \(X\) in terms of it eigenvalue decomposition \[ X=\sum_{i=1}^n\lambda_i q_i q^T_i. \] Then \[ \text{tr}(YX)=\text{tr}\left(Y\sum_{i=1}^n\lambda_i q_i q^T_i\right)\\ =\sum_{i=1}^n\lambda_i\text{tr}\left(Y q_i q^T_i\right)\quad (\text{tr}(A+B)=\text{tr}(A)+\text{tr}(B))\\ =\sum_{i=1}^n\lambda_iq_i^TYq_i\ge 0. \] Necessity proved.

Example Dual of a norm cone. Let \(||\cdot||\) be a norm on \(\mathbb R^n\). The dual of the associated cone \[ K=\{(x,t)\in \mathbb R^{n+1}\;|\; ||x||\le t\} \] is the cone defined by the dual norm \[ K^*=\{(u,v)\in \mathbb R^{n+1}|\space ||u||_*\le v\}, \] where the dual norm is given by \(||u||_* = \sup\{u^Tx\;|\; ||x|| \le 1\}\). The dual norm of the second norm is the second norm itself.

Dual cones satisfy several properties such as

  1. \(K^*\) is closed and convex.
  2. \(K_1 \subseteq K_2\) implies \(K_2^*\subseteq K_1^*\).
  3. If \(K\) has nonempty interior, then \(K^*\) is pointed.
  4. If the closure of \(K\) is pointed, then \(K^*\) has nonempty interior.
  5. If \(K\) is convex and closed, \(K^{**}=K\).

2.6.2 Dual generalized inequalities

Now suppose that the convex cone \(K\) is proper. Then it induces a generalized inequality \(\preceq_K\). Then its dual cone \(K^*\) is also proper, and therefore induces a generalized inequality.

  1. \(x\preceq_K y\) if and only if \(\lambda^T x\le \lambda ^T y\) for all \(\lambda \succeq_{K^*}0\)
  2. \(x\preceq_K y\) if and only if \(\lambda^T x< \lambda ^T y\) for all \(\lambda \succeq_{K^*}0\), \(\lambda \neq 0\)

2.6.3 Minimum and minimal elements via dual inequalities

We can use dual generalized inequalities to charaterize minimum and minimal elements of a (possitively nonconvex) set \(S\subseteq \mathbb R^n\) with respect to the generalized inequality induced by a proper cone \(K\).

Dual characterization of minimum element (Unique)

\(x\) is the minimum element of \(S\), with respect to the generalized inequality \(\preceq_K\), if and only if for all \(\lambda\succ_{K^*}0\), \(x\) is the unique minimizer of \(\lambda^Tz\) over \(z\in S\). Geometrically, this means that for any \(\lambda\succ_{K^*}0\), the hyperplane \[ \{z\;|\;\lambda^T(z-x)=0\} \] is strict supporting hyperplane to \(S\) at \(x\). (By strict supporting plane, we mean the hyperplane only intersects \(S\) at \(x\))

Dual characterization of minimal elements (Maybe many)

We now turn to a similar characterization of minimal elements. Here there is a gap between the necessary and sufficient conditions. If \(\lambda\succeq_{K^*}0\) and \(x\) minimizes \(\lambda^Tz\) over \(z\in S\), then \(x\) is minimal. The converse is ingeneral false.
