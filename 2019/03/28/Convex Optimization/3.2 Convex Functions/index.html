3.2 Operations that Preserve Convexity

We describe some operations that preserve convexity or concavity of functions, or allow us to construct new convex and concave functions.

3.2.1 Nonnegative weighted sums

a nonnegative weighted sum of convex functions, \[ f=w_1f_1+\dotsm+w_mf_m \] is convex.

These properties can be extended to infinite sums and integrals. For example, if \(f(x,y)\) is convex in x for each \(y\in \mathcal A\) and \(w(y)>0\)

The the function \(g\) defined as \[ g(x)=\int_\mathcal Aw(y)f(x,y)dy \] is convex.

3.2.2 Composition with an affine mapping

Suppose \(f:R^n \rightarrow R,A\in R^{n\times m}\) and \(b\in R^n\) Define \(g:R^m\rightarrow R\) by \[ g(x)=f(Ax+b) \] Then if \(f\) is convex, so is \(g\).

3.2.3 Pointwise maximum and supremum

If \(f_1\) and \(f_2\) are convex functions then their pointwise maximum \(f\), defined by \[ f(x)=\max\{f_1(x),f_2(x)\} \] with \(\dom f=\dom f_1\cap \dom f_2\) is also convex.

proof: if \(0\le \theta\le 1\) and \(x,y \in domf\), then \[ f(\theta x+(1-\theta)y)=\max\{f_1(\theta x+(1-\theta)y),f_2(\theta_1x+(1-\theta)y)\}\\ \le\max\{\theta f_1(x)+(1-\theta)f_1(y),\theta f_2(x)+(1-\theta)f_2(y)\}\\ \le\theta\max\{f_1(x),f_2(x)\}+(1-\theta)\max\{f_1(y),f_2(y)\}\\ =\theta f(x)+(1-\theta)f(y) \] ==Example== piecewise-linear functions is convex because every piecewise-linear function is affine. \[ f(x)=\max \{a_1^Tx+b_1,\dotsm,a_L^Tx+b_L\} \] The pointwise maximum property extends to pointwise supremum over an infinite set of convex functions. If for each \(y\in\mathcal A,f(x,y)\) is convex in \(x\), then the function \(g\) defined as \[ g(x)=\sup_{y\in\mathcal A}f(x,y) \] is convex. Here the domain of \(g\) is \[ \dom g=\{x|(x,y)\in \dom f\space \forall y\in\mathcal A,\sup_{y\in\mathcal A}f(x,y)\le \infty\} \] ==example== Support function of a set \[ S_C(x)=\sup\{x^Ty|y\in C\} \] is convex where \(C\subseteq \RR^n\) with \(C\neq \empty\)

==example== Distance to farthest point of a set \[ f(x)=\sup_{y\in C}||x-y|| \] is convex.

==Example== Least-square cost as a function of weights \[ g(w)=\inf_x \sum_{i=1}^n w_i(a_i^Tx-b_i)^2 \] ==example== Maximum eigenvalue of a symmetric matrix

The function \(f(X)=\lambda_{max}(X)\), with \(\dom f=S^m\) is convex. \[ f(X)=\sup\{y^TXy|\space||y||_2=1\} \] ==example== Norm of a matrix \(f(X)=||X||_2\), we have \[ f(X)=\sup\{u^TXv|\space ||u||_2=1,||v||_2=1\} \]

2.2.4 Composition

\[ f(x)=h(g(x)),\quad \dom f=\{x\in \dom g|g(x)\in \dom h\} \]

Scalar composition, the second derivative of the composition function \(f=h\circ g\) is given by \[ f''(x)=h''(g(x))g'(x)^2+h'(g(x))g''(x) \] that means if \(g\) is convex and h is convex and none decreasing, \(f\) is convex or \(h\) is convex and none increasing, \(g\) is concave.

Vector composition \[ f(x)=h(g(x))=h(g_1(x),\dotsm,g_k(x)) \] Then \[ f''(x)=g'(x)^T\nabla^2h(g(x))g'(x)+\nabla h(g(x))^Tg''(x) \] Then we can derive the following rules

\(f\) is convex if \(h\) is convex, \(h\) is none decreasing in each argument and \(g_i\) are convex or if \(h\) is convex, \(h\) is none increasing in each argument and \(g_i\) are concave

3.2.5 Minimization

We haven seen that the maximum or supremum of an arbitrary family of convex functions is convex. Here we consider minimum function.

If \(f\) is convex in (x, y) and C is convex nonempty set, then the function \[ g(x)=\inf_{y\in C}f(x,y) \] is convex in x provided \(g(x)\ge-\infty \space\forall x\)

proof

we have \(y_1,y_2\in C\) \[ g(\theta x_1+(1-\theta)x_2)=\inf_{y\in C} f(\theta x_1+(1-\theta)x_2,y)\\ \le f(\theta x_1+(1-\theta)x_2,\theta y_1+(1-\theta)y_2)\\ \le\theta f(x_1,y_1)+(1-\theta)f(x_2,y_2)\\ \le\theta g(x_1)+(1-\theta)g(x_2) \]

==Example== Schur complement

Suppose the quadratic function \[ f(x,y)=x^TAx+2x^TBy+y^TCy \] is convex in (x,y), which means \[ \begin{bmatrix} A & B\\B^T & C \end{bmatrix}\succeq 0 \] we can express \(g(x)=\inf _y f(x,y)\) as \[ g(x)=x^T(A-BC^{-1}B^T)x \] By the minimization rule, g(x) is also convex.

3.2.6 Perspective of a function

If \(f:R^n\rightarrow R\) , then the perspective of \(f\) is the function \(g: R^{n+1}\rightarrow R\) defined by \[ g(x,t)=tf(\frac{x}{t}) \] Then the perspective operation preserves convexity: if \(f\) is a convex function, then so is its perspective function \(g\).

3.3 The conjugate function

3.3.1 Definition and examples

Let \(f:R^n\rightarrow R\) . The function \(f^*:R^n\rightarrow R\) defined as \[ f^*(y)=\sup_{x\in dom f}(y^Tx-f(x)) \] is called the conjugate of the function \(f\). The ==domain== of the ==conjugate function== consists of \(y\in R^n\) for which the supremum is finite. We see immediately that \(f^*\) is convex since it is the pointwise supremum of a family of convex (affine) functions of y. That is true whether or not f is convex.

Let's start with some simple examples which allows us to derive an analytical expression for the conjugate of many common convex functions.

==Example==

Affine function. \(f(x)=ax+b\) As a function of x, \(yx-ax-b\) is bounded if and only if y=a, in which case it is a constant. Therefore, the domain of the convex function is a singleton \({a}\) and \(f^*(a)=-b\)

Negative logarithm. \(f(x)=-\log x\) with dom f is \(R_{++}\) . Then the function \(xy+\log x\) is unbounded above if \(y>0\) and reaches its maximum at \(x=-\frac{1}{y}\) . Then dom \(f^*=-R_{++}\) and \(f^*(y)=-1-\log(-y)\)

Exponential. \(f(x)=e^x\) . \(xy-e^x\) is unbounded if \(y<0\) . For \(y>0\), \(xy-e^x\) reaches its maximum at \(x=\log y\), we have \(f^*(y)=y\log y-y\) for \(dom f^*=R_{++}\)

Negative Entropy \(f(x)=x\log x\) . \(xy-x\log x\) is bounded. \(f^*(y)=e^{y-1}\)

3.3.2 Basic properties

Fenchel's inequality \[ f(x)+f^*(y)\ge x^Ty \] Conjugate of the conjugate

If \(f\) is convex and \(f\) is closed. Then \(f^{**}=f\)

Differentiable functions

Suppose \(f\) is convex and differentiable, with \(dom f= R^n\). Any maximizer \(x^*\) of \(y^Tx-f(x)\) satisfies \(y=\nabla f(x^*)\). Therefore, the conjugate function can be denoted as \[ f^*(y)=(x^*)^T\nabla f(x^*)-f(x^*) \] Scaling and composition with affine transformation

For \(a>0\) and \(b\in R\), the conjugate of \(g(x)=af(x)+b\) is \(g^*(y)=af^*(y/a)-b\)

Sums of independent functions

If \(f(u,v)=f_1(u)+f_2(v)\) where \(f_1\) and \(f_2\) are convex functions with conjugates \(f_1^*\) and \(f_2^*\) \[ f^*(w,z)=f_1^*(w)+f_2^*(z) \]

3.4 Quasiconvex functions

3.4.1 Definition and examples

A function \(f:R^n\rightarrow R\) is called quasiconvex if its domain and all its sublevel sets \[ S_\alpha=\{x\in dom f|\space f(x)\le \alpha\} \] for \(\alpha \in R\) are convex.

A function \(f:R^n\rightarrow R\) is called quasiconcave if its domain and all its superlevel sets \[ S_\alpha=\{x\in dom f|\space f(x)\ge \alpha\} \] for \(\alpha \in R\) are convex.

A function that is both quasiconvex and quasiconcave is called quasilinear. If a function \(f\) is quasilinear, then its domain and every level set \(\{x|f(x)=\alpha\}\) is convex.

==Example== Logarithm is quasiconvex and quasiconcave and therefore is quasilinear.

==Example== Consider \(f:R^2\rightarrow R\) with \(\dom f =\RR_+^2\) and \(f(x_1, x_2)=x_1x_2\) . This function is neither convex nor concave since its hessian matrix \[ \nabla^2f(x)= \begin{bmatrix} 0 & 1\\ 1 & 0 \end{bmatrix} \] However the function is quasiconcave since the superlevel sets \[ \{x\in \RR_+^2| x_1x_2\ge \alpha\} \] are convex sets for all \(\alpha\)

3.4.2 Basic Properties

A function \(f\) is quasiconvex if and only if \(\dom f\) is convex and for any \(x,y \in \dom f\) and \(0\le \theta\le 1\) , \[ f(\theta x+(1-\theta) y) \le \max \{f(x),f(y)\} \] That means the value of the fdoes not exceed the maximum of its values of the endpoints.

\(f\) is quasiconvex if and only if its restriction to any line intersecting its domain is quasiconvex.

==Quasiconvex Functions on R==

A continuous function \(f:\RR\rightarrow\RR\) is quasiconvex if and only if at least one of the following condition holds:

  1. \(f\) is nondecreasing
  2. \(f\) is nonincreasing
  3. there is a point \(c\in \dom f\) such that for \(t<c\) (and \(t\in \dom f\)), f is nonincreasing, and for \(t\ge c\) (and \(t\in \dom f\)), \(f\) is nondecreasing.

The point c can be chosen as any point which is a global minimizer of \(f\) .

3.4.3 Differentiable quasiconvex functions

==First order conditions==

Suppose \(f: \RR^n\rightarrow \RR\) is differentiable. Then \(f\) is quasiconvex if and only if \(\dom f\) is convex and for all \(x,y\in \dom f\) \[ f(y)\le f(x)\implies \nabla f(x)^T(y-x)\le 0 \] ==Second order conditions==

Suppose \(f\) is twice differentiable. If \(f\) is quasiconvex, then for all \(x \in \dom f\), and for all \(y \in R^n\) we have \[ y^T\nabla f(x)=0\implies y^T\nabla^2f(x) y\ge 0 \]

3.4.4 Operations that preserve quasiconvexity

==Nonnegative weighted maximum== \[ f=\max\{w_1f_1,\dotsm,w_mf_m\} \] The property can be expanded to general pointwise supremum. \[ f(x)=\sup_{y\in C}(w(y)g(x,y)) \] where \(w(y)\ge 0\) and \(g(x,y)\) is quasiconvex in x for each y.

==Composition==

If \(g:R^n \rightarrow R\) is quasiconvex and \(h:R\rightarrow R\) is non decreasing, then $f=g h $ is quasiconvex.

==Minimization==

If \(f(x,y)\) is quasiconvex jointly in \(x,y\) and C is a convex set. Then the function \[ g(x)=\inf_{y\in C}f(x,y) \] is quasiconvex.

3.4.5 Representation via family of convex functions

In the sequel, it will be convenient to represent the sublevel sets of a quasiconvex function via inequalities of convex functions. We seek a family of convex functions \(\phi(t):R^n\rightarrow R\) indexed by \(t\in R\) \[ f(x)\le t \iff \phi_t(x)\le 0 \]

3.5 Log-concave and log-convex functions

3.5.1 Definition

A function \(f: R^n\rightarrow R\) is logarithmically concave or log-concave if \(f(x)>0\) for all \(x\in dom f\) and \(\log f\) is concave. It is said to be logarithmically convex or log-convex if \(\log f\) is convex. Thus \(f\) is log-convex if and only if \(1/f\) is log-concave. It is convenient to allow \(f\) to take on the value zero, in which case we take \(\log f(x)=-\infty\). In the case we say \(f\) is log-concave if the extended-value function \(\log f\) is concave.

We can express log-concave function directly without logarithms: a function \(f: R^n\rightarrow R\) with convex domain and \(f(x)>0\) for all \(x\in dom f\) is log-concave if and only if for all \(x,y \in dom f\) and \(0\le \theta\le 1\) , we have \[ f(\theta x+(1-\theta) y)\ge f(x)^\theta f(y)^{1-\theta} \] From the composition rules we know that \(e^h\) is convex if \(h\) is convex since the function \(e^x\) is convex and nondecreasing and function \(h\) is convex. So a log-convex function is convex. Similarly , a nonnegative concave function is log-concave since \(\log x\) is concave and nondecreasing. It is also clear that a log-convex function is quasiconvex and a log-concave function is quasiconcave since the logarithm is monotone increasing.

3.5.2 Properties

Twice differentiable log-convex/concave functions.

Suppose \(f\) is twice differentiable, with \(dom f\) is convex,so \[ \nabla^2 \log f(x)=\frac{1}{f(x)}\nabla^2 f(x)-\frac{1}{f(x)^2}\nabla f(x)\nabla f(x)^T \] we conclude that \(f\) is log-convex if and only if for all \(x\in dom f\) \[ f(x)\nabla^2 f(x)\succeq \nabla f(x)\nabla f(x)^T \] and log-concave if and only if for all \(x\in dom f\) \[ f(x)\nabla^2 f(x)\preceq \nabla f(x)\nabla f(x)^T \] ==Multiplication, addition and integration==

Log-convexity and log-concavity are closed under multiplication and positive scaling. For example, if \(f\) and \(g\) are log-concave, then so is the pointwise product. \(h(x)=f(x)g(x)\)

Sum of log-concave functions is not in general log-concave. Log-convexity, however, preserved under sums.

In some special case log-concavity is preserved by integration. If \(f: R^n \times R^m \rightarrow R\) is log-concave, then \[ g(x)=\int f(x,y)dy \] is log-concave function of \(x\) on \(R^n\)

Convolution: if \(f\) and \(g\) are log-concave on \(R^n\), then so is the convolution \[ (f*g)(x)=\int f(x-y)g(y)dy \]

3.6 Convexity with respect to generalized inequalities

We now consider generalizations of notions of monotonicity and convexity, using generalized inequalities instead of the usual ordering on R.

3.6.1 Monotonicity with respect to a generalized inequality

Suppose \(K\subseteq R^n\) is a proper cone with associated generalized inequality \(\preceq_K\). A function \(f:R^n\rightarrow R\) is called K-nondecreasing if \[ x\preceq_K y\implies f(x)\le f(y) \] and K-increasing if \[ x\preceq_K y, x\neq y \implies f(x)<f(y) \] K-nonincreasing and K-decreasing functions are defined in a similar way.

Gradient conditions for monotonicity

A differentiable function \(f\), with convex domain, is K-nondecreasing if and only if \[ \nabla f(x)\succeq_{K^*}0 \] The function is increasing if \[ \nabla f(x)\succ_{K^*}0 \]

3.6.2 Convexity with respect to a generalized inequality

Suppose \(K\subseteq R^n\) is a proper cone with associated generalized inequality \(\preceq_K\). A function \(f:R^n\rightarrow R^n\) is called K-convex if for all \(x,y\) and \(0\le \theta\le 1\) \[ f(\theta x+(1-\theta)y) \preceq_K \theta f(x)+(1-\theta)f(y) \] Strict convex if \[ f(\theta x+(1-\theta)y) \prec_K \theta f(x)+(1-\theta)f(y) \]
